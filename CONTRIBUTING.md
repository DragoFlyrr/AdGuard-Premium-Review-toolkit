# Contributing to AdGuard Resource Hub

Thank you for your interest in contributing!

## How to Contribute

### Reporting Issues

If you find broken links or outdated information:
1. Open a GitHub issue
2. Describe the problem
3. Provide the correct information if known

### Adding Content

We welcome:
- New guides and tutorials
- Updated comparisons
- Additional resources
- Translation contributions

### Content Guidelines

- Use clear, concise language
- Link only to official sources
- Keep information up to date
- Respect trademarks and copyrights

### Code of Conduct

- Be respectful and constructive
- No spam or self-promotion
- Focus on helping users

## Questions?

Open an issue for any questions about contributing.
