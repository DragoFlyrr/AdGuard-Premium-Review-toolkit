## üíª Windows Installation

### Step 1: Download
1. Visit [adguard.com/download](https://adguard.com/en/download.html)

3. Save the installer (AdGuardInstaller.exe)

### Step 2: Install
1. Run the downloaded installer
2. Follow the setup wizard

1. Launch AdGuard from Start Menu
2. Complete the onboarding tutorial
3. Enable protection
4. Configure your preferred filter lists

## üîß Post-Installation Tips

### Essential Settings
- [ ] Enable all protection modules
- [ ] Update filter lists
- [ ] Configure whitelist for trusted sites
- [ ] Enable HTTPS filtering (recommended)

### Troubleshooting

**Issue:** Ads still showing
- **Solution:** Update filter lists, clear browser cache

**Issue:** Website broken
- **Solution:** Add to whitelist or report to filter maintainers

**Issue:** Certificate warnings
- **Solution:** Install AdGuard certificate for HTTPS filtering
